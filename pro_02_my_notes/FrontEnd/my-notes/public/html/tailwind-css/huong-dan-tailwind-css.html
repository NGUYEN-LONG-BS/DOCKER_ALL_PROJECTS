Hướng Dẫn Sử Dụng Tailwind CSS

Tìm hiểu cách sử dụng Tailwind CSS với các ví dụ cụ thể và chi tiết

1. Màu Sắc và Nền

Tailwind cung cấp hệ thống màu sắc với các sắc độ từ 50 đến 900, áp dụng cho nền, chữ, viền, v.v.

Khối màu xanh với chữ trắng
Khối màu đỏ nhạt với chữ xám đậm
        
<div class="bg-blue-500 text-white p-4 rounded">
  Đây là một khối màu xanh với chữ trắng
</div>
<div class="bg-red-200 text-gray-800 p-4 mt-2 rounded">
  Đây là một khối màu đỏ nhạt với chữ xám đậm
</div>

Giải thích: bg-blue-500 đặt nền xanh, text-white đặt chữ trắng, p-4 thêm padding 1rem, rounded bo góc, mt-2 thêm margin-top 0.5rem.

2. Kích Thước và Khoảng Cách

Quản lý padding, margin, width, height với đơn vị rem (1rem = 16px).

Khối rộng 256px, cao 128px, căn giữa, padding 16px.
Khối chiếm toàn bộ chiều rộng, cao 12rem, padding dọc 32px.
        
<div class="w-64 h-32 bg-green-300 mx-auto p-4 my-4">
  Khối này có chiều rộng 256px, cao 128px, căn giữa, và padding 16px.
</div>
<div class="w-full h-48 bg-gray-100 py-8">
  Khối này chiếm toàn bộ chiều rộng và cao 12rem, padding dọc 32px.
</div>

Giải thích: w-64 h-32 đặt kích thước, mx-auto căn giữa, w-full h-48 chiếm toàn bộ chiều rộng và cao 12rem.

3. Flexbox và Grid

Tạo bố cục dễ dàng với Flexbox và Grid.

Khối con căn giữa
Cột 1
Cột 2
Cột 3
        
<!-- Flexbox -->
<div class="flex justify-center items-center h-64 bg-gray-200">
  <div class="bg-blue-500 text-white p-4 rounded">Khối con căn giữa</div>
</div>

<!-- Grid -->
<div class="grid grid-cols-1 sm:grid-cols-3 gap-4 p-4 bg-gray-100">
  <div class="bg-blue-300 p-4 rounded">Cột 1</div>
  <div class="bg-blue-400 p-4 rounded">Cột 2</div>
  <div class="bg-blue-500 p-4 rounded">Cột 3</div>
</div>

Giải thích: flex justify-center items-center căn giữa nội dung trong Flexbox, grid-cols-3 tạo lưới 3 cột trên màn hình lớn, gap-4 đặt khoảng cách 1rem.

4. Typography

Điều chỉnh kích thước chữ, độ đậm, căn chỉnh với các lớp tiện ích.

Tiêu đề lớn

Đây là đoạn văn bản mô tả.

        
<div class="p-4 bg-gray-100">
  <h1 class="text-3xl font-bold text-center text-blue-600">Tiêu đề lớn</h1>
  <p class="text-lg font-semibold text-left text-gray-700 mt-2">Đây là đoạn văn bản mô tả.</p>
</div>

Giải thích: text-3xl đặt kích thước chữ lớn, font-bold chữ đậm, text-center căn giữa tiêu đề.

5. Responsive Design

Sử dụng tiền tố sm:, md:, lg:, xl: để áp dụng lớp theo breakpoint.

Responsive Text
Khối 1
Khối 2
Khối 3
        
<div class="text-center sm:text-left md:text-2xl lg:bg-blue-600 lg:text-white p-4">
  Responsive Text
</div>
<div class="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 gap-4 p-4">
  <div class="bg-blue-200 p-4 rounded">Khối 1</div>
  <div class="bg-blue-300 p-4 rounded">Khối 2</div>
  <div class="bg-blue-400 p-4 rounded">Khối 3</div>
</div>

Giải thích: sm:text-left căn trái trên màn hình ≥640px, md:grid-cols-3 tạo lưới 3 cột trên màn hình ≥768px.

6. Hover, Focus, Active

Thêm hiệu ứng tương tác với tiền tố hover:, focus:, active:.

        
<button class="bg-blue-500 hover:bg-blue-700 focus:bg-blue-800 active:bg-blue-900 text-white p-2 rounded focus:ring-2 focus:ring-blue-300">
  Nhấn tôi
</button>
<input type="text" class="border-2 border-gray-300 p-2 rounded focus:border-blue-500 focus:ring-2 focus:ring-blue-200" placeholder="Nhập text...">

Giải thích: hover:bg-blue-700 đổi màu nền khi di chuột, focus:ring-2 thêm viền hiệu ứng khi focus.

7. Ví dụ Tổng Hợp: Card Responsive

Tạo một card responsive với Tailwind CSS.

Card Tiêu Đề

Đây là nội dung mô tả của card, sử dụng Tailwind CSS để định dạng.

        
<div class="bg-white p-6 rounded-lg shadow-lg w-full max-w-sm sm:max-w-md md:max-w-lg mx-auto">
  <h2 class="text-2xl font-bold text-center text-gray-800 mb-4">Card Tiêu Đề</h2>
  <p class="text-gray-600 mb-4">Đây là nội dung mô tả của card, sử dụng Tailwind CSS để định dạng.</p>
  <button class="bg-blue-500 hover:bg-blue-600 text-white px-4 py-2 rounded w-full sm:w-auto">
    Nhấn vào đây
  </button>
</div>

Giải thích: Card sử dụng max-w-sm sm:max-w-md md:max-w-lg để responsive, shadow-lg thêm bóng, hover:bg-blue-600 cho hiệu ứng nút.

8. Mẹo Bổ Sung

  • Tái sử dụng classes: Tạo component với @layer components để tái sử dụng nhóm lớp.
  • Debug nhanh: Sử dụng Tailwind Play để thử nghiệm giao diện.
  • VS Code: Cài extension Tailwind CSS IntelliSense để gợi ý lớp nhanh.
        
@layer components {
  .card {
    @apply bg-white p-6 rounded-lg shadow-lg;
  }
}

Giải thích: Định nghĩa component .card để áp dụng nhanh các lớp lặp lại.
