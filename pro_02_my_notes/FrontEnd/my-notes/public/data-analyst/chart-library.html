Thư viện React Chart Components Miễn Phí

Dưới đây là một số thư viện và dự án mã nguồn mở mà bạn có thể sử dụng các React chart components miễn phí, bao gồm các component .tsx cho TypeScript:

  • Recharts

    Link: https://recharts.org

    Đặc điểm: Dễ sử dụng, hỗ trợ nhiều loại biểu đồ như bar, line, pie, radar.

  • Chart.js với react-chartjs-2

    Link: https://www.chartjs.org/

    Đặc điểm: Hỗ trợ TypeScript và nhiều loại biểu đồ khác nhau.

    Package: react-chartjs-2

  • Victory

    Link: https://formidable.com/open-source/victory/

    Đặc điểm: Hoạt động tốt với TypeScript và cung cấp các biểu đồ tương tác mạnh mẽ.

  • Nivo

    Link: https://nivo.rocks/

    Đặc điểm: Hỗ trợ TypeScript và dễ dàng tùy chỉnh.

  • React-vis

    Link: https://uber.github.io/react-vis/

    Đặc điểm: Thiết kế đẹp và dễ tích hợp với React.

Cài đặt Thư viện

Các thư viện trên đều miễn phí và hỗ trợ TypeScript, bạn có thể sử dụng chúng trực tiếp trong dự án .tsx của mình. Bạn chỉ cần cài đặt thông qua npm hoặc yarn, ví dụ:

npm install recharts

Hoặc nếu bạn sử dụng react-chartjs-2:

npm install react-chartjs-2 chart.js

Sau đó, bạn có thể import và sử dụng các biểu đồ trong các component .tsx.
